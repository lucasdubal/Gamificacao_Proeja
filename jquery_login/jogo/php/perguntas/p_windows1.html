Windows - Copiar, Colar, Recortar (Ctrl C, Ctrl V, Ctrl X)

Agora, vamos aprender alguns conceitos básicos de Windows. Você sabe o significado das teclas Ctrl + C, Ctrl + V e Ctrl + X?? Se sim, ótimo; se não, é simples.

Quando você está no computador, e seleciona um arquivo - uma imagem, um arquivo do Word, qualquer coisa - e aperta Ctrl + C, você estará COPIANDO esse arquivo.

Logo, você poderá ir em outro local do seu computador e colar o arquivo lá, tendo o arquivo agora em dois locais, pois você copiou. Ok...Ctrl + C você copia.

E como COLA? Com o comando Ctrl + V!!

Você agora sabe copiar um arquivo e colar em outro lugar...mas e se você quiser movê-lo de um lugar para outro??? Você quer RECORTÁ-LO, no caso.

Para mover/recortar um arquivo, utilizamos o Ctrl + X. Para colar o arquivo em outro lugar, o mesmo Ctrl + V de antes.

Abra seu Windows Explorer e pratique!!!

Você pode associar Ctrl + C (copiar/cópia), porém Ctrl + X e Ctrl + V não são tão fáceis para associar. Por isso, treine e pratique!!
