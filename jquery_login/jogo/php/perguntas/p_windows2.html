Navegadores - Ctrl T, Ctrl N, Ctrl D, Ctrl H

Abordando o assunto navegadores, é importante ter conhecimento de alguns atalhos de teclado, a exemplo dos citados no título.

É interessante ir praticando cada um para ver o que acontece. Então, abra seu navegador de internet (Google Chrome, Firefox, Internet Explorer, enfim!) e segue lá!!

*Ctrl+T: Abre uma nova aba. Perceba que uma aba é diferente de uma janela!

*Ctrl+N: Esse sim abre uma janela. Compare com o anterior e veja a diferença!

*Ctrl+H: Abre o histórico(páginas visitadas) do seu navegador. H de Histórico!

*Ctrl+D: Adiciona um site aos Favoritos! Como o nome diz, Favoritos é uma área onde ficam os sites que você mais gosta. Colocando um site lá, você consegue ter acesso a eles de forma mais fácil e rápida. Dica: FavoDitos!! hehe

Após ter praticado as teclas ensinadas, responda a questão abaixo:

O atalho Ctrl+T abre uma nova aba, o atalho Ctrl+N uma nova janela; o histórico pode ser acessado via Ctrl+H, e adicionar aos favoritos acontece via Ctrl+F.
